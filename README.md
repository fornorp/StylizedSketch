# StylizedSketch
Stylized Sketch Synthesis Using Diffusion Priors and Structural Guidance.

This project is built upon [Attention Distillation](https://github.com/xugao97/AttentionDistillation) and you need to install [DiffVG](https://github.com/BachiLi/diffvg). 

Execute run_batch.py to test our method.
